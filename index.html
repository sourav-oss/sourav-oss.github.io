AMATrack (ESTEPS) – Smart Transit Management Platform

Description: ESTEPS AMATrack is a smart public transit management platform developed for the Puerto Rico Metropolitan Bus Authority. It enables efficient route planning, live vehicle tracking, driver shift management, and passenger route suggestions based on real-time transit data. One of its standout features is Call & Travel, allowing passengers to schedule trips in advance via the platform.

  • Developed a responsive and dynamic web interface using React.js for both admin and passenger portals.
  • Built reusable modules such as real-time map tracking, ETA display, driver allocation, and trip scheduling using React's lifecycle methods and hooks.
  • Implemented interactive route visualizations and location-based features using external mapping services.
  • Two separate mobile applications were developed in Java (Android Studio)—one for passengers and another for drivers.
  • Improved commuter experience through accurate live updates, schedule notifications, and pre-booking capabilities.

Skills: React.js, Android Studio, Web Application Development, Mobile Applications, Responsiveness, Geological Mapping
